# yolox_onnx_sample
ONNXRuntimeでYOLOXを動かすサンプル

## 準備
- `pip install onnxruntime opencv-python python-box numba`
- [YOLOX-ONNXRuntime in Python](https://yolox.readthedocs.io/en/latest/demo/onnx_readme.html)からWeightsをダウンロードしmodelsに置く
